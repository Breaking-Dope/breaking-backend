## Core Functions
- OAuth2.0 회원가입 및 로그인 (JWT, Refresh Token, Spring Security)
- 기본 SNS 기능 (게시글 CRUD, 좋아요, 북마크, 게시글 숨김, )
- 게시글 피드 (필터, 정렬, 커서 페이징)
- 게시글 거래 기능 (포인트 충전, 게시글 구매, 구매 기록)
- 미디어 처리 (영상 썸네일, 워터마크, 압축)
